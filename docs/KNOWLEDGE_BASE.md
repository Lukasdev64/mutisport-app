# Base de Connaissance MultiSport Platform

## Tarifs et Abonnements

### Abonnement Gratuit
- **Prix :** 0€ / mois
- **Fonctionnalités :**
  - Jusqu'à 3 tournois
  - Gestion basique des joueurs
  - Formats standards de tournois

### Abonnement Pro
- **Prix Mensuel :** 9.99€ / mois
- **Prix Annuel :** 99.99€ / an (environ 20% de réduction)
- **Fonctionnalités :**
  - Tournois illimités
  - Statistiques avancées
  - Support prioritaire
  - Personnalisation complète
  - Export des données
  - Badge Pro sur le profil

## Fonctionnalités Principales

### Gestion de Tournois
- Création rapide de tournois (Tennis, Basket, etc.)
- Génération automatique des tableaux (Brackets)
- Gestion des scores en temps réel

### Gestion des Joueurs
- Profils joueurs détaillés
- Suivi des performances
- Historique des matchs

### Application Mobile
- L'application est une PWA (Progressive Web App) installable sur iOS et Android.
- Navigation optimisée pour mobile avec une barre de navigation en bas.

## Support
